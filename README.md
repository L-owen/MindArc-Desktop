# 闪念 MindArc

> 心有所念，落笔成篇

心中念头起时，随手记之。数据尽在本地，不经云端，不涉账号，不留痕迹。

## 特性

- **随手记** — 底栏常驻，念起即书，不费吹灰
- **轻重缓急** — 急要、重要、紧急、闲记，四档分明
- **分门别类** — 预置标签，亦可自创，条理井然
- **开卷即来** — 记忆上次所阅之卷，再启自动续读
- **时辰提醒** — 设定时刻，准时相告，可循环往复
- **落笔有痕** — 每次保存皆留备份，万无一失
- **明暗随心** — 赛博暗色为底，亦可切换明光

## 隐私

- 数据唯存一本地 `.json` 文件，由君自择存放之所
- 不联网、不统计、不窥探，念念皆私
